# Short Video Script Prompt

Please generate a 60-90 second short video script based on the topic below.

## Requirements

- Audience: SMB brand owners.
- The first 3 seconds must directly hit a pain point.
- Avoid jargon. If jargon is necessary, translate it into business-owner language.
- Structure: pain point -> judgment -> example -> checklist -> action suggestion.
- Tone: calm, professional, slightly sharp.
- Do not promise ROI.
- End with a soft CTA, such as asking viewers to get a checklist or send a private message for diagnosis.
